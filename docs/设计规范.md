# 曦月笔记 — 设计规范

## 设计理念
**科技感 + 未来感**，深色为主，紫色为魂。磨砂玻璃、微光效、呼吸动画。

## 阴影与光效

### 卡片阴影
```
默认: 0 4px 20px rgba(0, 0, 0, 0.3)
悬浮: 0 8px 32px rgba(124, 58, 237, 0.15)
```

### 曦筑光晕
```
box-shadow: 0 0 20px rgba(168, 85, 247, 0.3),
            0 0 40px rgba(168, 85, 247, 0.1)
border: 1px solid rgba(168, 85, 247, 0.5)
```

### 按钮微光（hover）
```
box-shadow: 0 0 15px rgba(124, 58, 237, 0.4)
transition: all 0.3s ease
```

## 动效规范

| 元素 | 动效 | 时长 |
|------|------|------|
| 页面切换 | 淡入淡出 + 轻微上移 | 0.3s |
| 按钮 hover | 背景色过渡 + 微光 | 0.2s |
| 卡片 hover | 上浮 2px + 阴影加深 | 0.3s |
| 弹窗出现 | 缩放 + 淡入 | 0.25s |
| 加载状态 | 紫色脉冲动画 | 循环 |
| 图谱节点 | 呼吸灯效果 | 2s 循环 |

### 缓动函数
- 默认：`cubic-bezier(0.4, 0, 0.2, 1)` (ease-in-out)
- 弹性：`cubic-bezier(0.34, 1.56, 0.64, 1)` (弹性进入)

---

## 磨砂玻璃效果 (Glassmorphism)

```css
background: rgba(30, 27, 46, 0.7);
backdrop-filter: blur(16px) saturate(120%);
border: 1px solid rgba(168, 85, 247, 0.15);
```

应用于：侧边栏、弹窗、顶部导航、底部导航
