# 曦月笔记 — 需求规格说明书

## 概述
曦月笔记是一款面向个人用户的笔记应用，支持安卓手机（PWA安装）和网页浏览器双端使用。数据存储在设备本地（IndexedDB），无需联网、无需登录、打开即用。

---

## 功能需求

### 1. 用户系统（无需注册）
- **无需账号**：打开即可使用，所有数据存储在设备本地
- **应用锁**：笔记可加锁，加锁后不可编辑，需解锁后才能修改

### 2. 笔记管理
#### 2.1 新建笔记
- 每篇笔记包含：
  - **主标题**（选填）：笔记的主要标题
  - **副标题**（选填）：笔记的次要标题
  - **正文内容**：富文本格式
- 新建后自动保存到本地数据库

#### 2.2 修改笔记
- 点击已有笔记进入编辑模式
- 修改后自动保存

#### 2.3 删除笔记
- 删除前弹出确认对话框
- 删除后从列表和本地数据库中移除

#### 2.4 笔记列表
- 卡片式布局展示所有笔记
- 显示主标题、副标题摘要、标签、更新时间
- 按更新时间倒序排列

### 3. 富文本编辑功能
编辑器中提供以下格式工具栏：

| 功能 | 说明 |
|------|------|
| 加粗 | 选中文字加粗 |
| 下划线 | 选中文字加下划线 |
| 斜体 | 选中文字倾斜 |
| 删除线 | 选中文字加删除线 |
| 字体大小 | 支持 10 号 ~ 60 号，通过滑块或下拉选择 |
| 字体颜色 | 预设 7 种颜色：赤(#EF4444)、橙(#F97316)、黄(#EAB308)、绿(#22C55E)、蓝(#3B82F6)、紫(#A855F7)、黑(#FFFFFF) |
| 左对齐 | 段落靠左对齐 |
| 居中 | 段落居中对齐 |
| 右对齐 | 段落靠右对齐 |
| 自动编号 | 有序列表，换行自动递增编号 |
| 待办事项 | 可勾选/取消勾选的清单项 |
| 插入图片 | 从本地选择图片上传到云端，插入编辑器 |
| 插入表格 | 插入可编辑的表格 |

### 4. 标签系统
- **创建标签**：标签名称 + 颜色（赤橙黄绿蓝紫黑 7 选 1）
- **编辑标签**：修改标签名称或颜色
- **删除标签**：删除后从所有笔记中移除该标签
- **打标签**：在笔记编辑时可选择一个或多个标签
- **按标签筛选**：笔记列表页可按标签过滤显示

### 5. 日历视图
- 以月历形式展示
- 每个日期格子内显示当天笔记的：
  - 主标题（小字显示）
  - 标签色点
- 点击某个日期可查看该日所有笔记
- 支持切换月份

### 6. 搜索功能
- 搜索范围：笔记主标题 + 副标题 + 正文内容
- 搜索结果高亮显示匹配关键词
- 点击搜索结果跳转到对应笔记

### 7. 数据同步
- 同一账号在不同设备登录，数据自动同步
- 同步内容：笔记、标签、链接关系、用户设置
- 离线时仍可查看已缓存的笔记

### 8. 双向链接
#### 8.1 创建链接
- 在编辑器中输入 `@` 符号，触发笔记主标题搜索弹窗
- 从列表中选择目标笔记后，自动插入可点击链接
- 链接显示为目标笔记的主标题

#### 8.2 反向链接
- 打开任意笔记时，在底部或侧边栏显示「反向链接」
- 列出所有链接到当前笔记的其他笔记
- 每条反向链接可点击跳转

### 9. 知识图谱
- 力导向网状图，可视化笔记之间的链接关系
- 节点：代表一篇笔记，显示主标题
- 连线：代表笔记之间存在链接关系
- **核心枢纽识别**：连线多的笔记节点更大更亮
- **孤岛识别**：没有连线的笔记显示在边缘
- **标签筛选**：下拉选择标签，仅显示相关笔记的网络
- **节点交互**：点击节点跳转到对应笔记

### 10. 曦筑
#### 10.1 笔记置顶
- 笔记列表页中，长按或点击置顶图标可「曦筑」笔记
- 曦筑的笔记始终排在最前面（无论排序方式）
- 曦筑笔记有视觉区分（紫色光晕边框）

#### 10.2 曦筑仪表板
- 独立页面，集中展示所有曦筑笔记的卡片
- 卡片上显示：
  - 主标题
  - 标签
  - 更新时间
  - 待办事项（可直接勾选，无需进入笔记）
- 笔记正文中的待办项勾选状态实时同步

#### 10.3 智能曦筑
- 用户可在设置中选择最多 3 个标签
- 系统自动将含有这些标签的笔记加入曦筑仪表板
- 手动曦筑 + 智能曦筑的结果合并展示

### 11. 每日随机回顾
- 「随机漫步」入口按钮
- 点击后随机抽取一条旧笔记，以闪卡形式展示
- 支持按标签限定随机范围
- 展示笔记标题、摘要
- 可点击进入完整笔记，或点击「再来一篇」换一条

## 非功能需求

### 平台支持
- 网页端：Chrome、Edge、Firefox 等主流浏览器
- 安卓端：通过 PWA 安装到手机桌面

### 性能
- 首页加载时间 < 3 秒
- 笔记保存响应 < 1 秒
- 图片上传支持进度显示

### 安全
- 数据纯本地存储（IndexedDB），无需登录、无需联网
- Android 端数据受应用沙箱保护，卸载 App 时自动清除

### UI 风格
- 科技感、未来感
- 深色主题背景
- 紫色主色调
- 磨砂玻璃效果面板
