# 曦月笔记 — 部署指南

曦月笔记是纯前端 PWA 应用，无需后端服务器、无需数据库、无需 Firebase。只需将静态文件部署到任意托管服务即可。

---

## 方式一：Vercel（推荐，免费）

### 1. 推送代码到 GitHub
1. 在 GitHub 创建仓库（如 `xiyue-notes`）
2. 推送项目：
```bash
git init
git add .
git commit -m "曦月笔记 v1.0"
git branch -M main
git remote add origin https://github.com/你的用户名/xiyue-notes.git
git push -u origin main
```

### 2. Vercel 部署
1. 打开 https://vercel.com/ ，用 GitHub 账号登录
2. 点击「New Project」→ 导入 GitHub 仓库
3. **无需配置环境变量**（纯本地应用，无后端）
4. 框架自动检测为 Vite → 点击「Deploy」
5. 1-2 分钟后获得公网 URL（如 `https://xiyue-notes.vercel.app`）

### 3. 后续更新
每次 `git push` 到 main 分支，Vercel 自动重新部署。

---

## 方式二：GitHub Pages（免费）

### 1. 配置构建
在项目根目录创建 `.github/workflows/deploy.yml`（略，可参考 Vite 官方文档）。

### 2. 注意
GitHub Pages 需要配置 SPA 回退（404 页面重定向到 index.html）。推荐使用 Vercel，已内置 SPA 支持（项目根目录的 `vercel.json` 已配置路由重写）。

---

## 本地开发

```bash
npm install        # 安装依赖
npm run dev        # 启动开发服务器 → http://localhost:5173
npm run build      # 构建生产版本
npm run preview    # 本地预览生产构建
```

## PWA 安装

### Android（Chrome）
1. 手机 Chrome 打开部署后的 URL
2. 地址栏右侧出现安装图标（或弹出安装横幅）
3. 点击「安装」→ 桌面出现曦月笔记图标
4. 安装后支持离线使用，数据受系统保护

### iOS（Safari）
1. Safari 打开 URL
2. 点击底部「分享」按钮
3. 滑动找到「添加到主屏幕」
4. 命名后点击「添加」

### 桌面（Chrome/Edge）
1. 地址栏右侧出现安装图标
2. 点击安装 → 应用窗口独立运行

---

## 离线使用

安装 PWA 后，应用的所有静态资源（HTML/JS/CSS/图标）已缓存到设备本地。断网情况下：
- ✅ 可以打开应用
- ✅ 查看、编辑笔记（数据存在本地 IndexedDB）
- ❌ 无法访问公网 URL（需先安装 PWA）

---

## 数据安全说明

- 数据存储在浏览器 IndexedDB 中，无需网络
- 应用启动时自动请求**永久存储权限**（防止系统自动清理）
- 已安装的 PWA 自动获得存储保护
- 建议定期使用设置页的「导出」功能备份数据
- 更换手机：旧手机导出 → 新手机导入
