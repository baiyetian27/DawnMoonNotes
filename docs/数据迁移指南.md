# 曦月笔记 — 数据迁移指南

## 适用场景

更换手机、清除浏览器数据、从测试版升级到正式版时，可使用本指南迁移数据。

## 导出数据（旧手机）

1. 打开曦月笔记
2. 进入「设置」页面（底部导航最右侧 ⚙️ 图标）
3. 点击「导出全部数据（JSON）」
4. 浏览器会下载一个文件：`曦月笔记备份_2026-06-24.json`
5. 将该文件发送到新手机（通过微信、钉钉、蓝牙、数据线等方式）

> **注意**：备份文件包含所有笔记内容、标签、链接关系和图片。图片以 base64 编码存储在 JSON 中，文件可能较大。

## 导入数据（新手机）

1. 在新手机上安装曦月笔记（通过浏览器安装 PWA）
2. 进入「设置」页面
3. 点击「从备份文件导入」
4. 选择从旧手机传来的 `曦月笔记备份_*.json` 文件
5. 系统会弹出确认弹窗（⚠️ 提醒：导入将覆盖当前数据）
6. 点击「确认导入」
7. 导入完成后显示统计信息：笔记 X 条、标签 X 条、链接 X 条

## 注意事项

- 导入会**清空**新设备上所有现有数据，请确认无误后再操作
- 备份文件为 JSON 格式，可以用文本编辑器打开查看（图片部分是 base64，很长）
- 建议定期导出备份，保存在电脑或云盘中
- 如果备份文件过大（超过 50MB），说明图片较多，可使用 Chrome 桌面版执行导入

## 技术说明

曦月笔记所有数据存储在浏览器的 IndexedDB 中，数据文件位置因浏览器和操作系统而异：

- **Windows Chrome**：`%LocalAppData%\Google\Chrome\User Data\Default\IndexedDB\`
- **Android Chrome**：手机内部存储 → `Android/data/com.android.chrome/`
- **PWA 安装版**：存储在 PWA 的独立存储空间中

如果浏览器缓存被清除，数据将丢失。请务必定期导出备份。
